Photopea Actions Lib

Open photopeaActionsLib.jsx

HOW USE?

STEP 1. Open photopeaActionsLib.jsx (orange button), select all library code and copy (Ctrl+C)

STEP 2. Open Photopea. Open File - Script.

STEP 3. Clear all code in script window. Insert library code (Ctrl+V) in script window.

STEP 4. Save library (as example, with name "photopeaActionsLib.jsx").

STEP 5. Close window. Check in the Edit - Locale Storage... that the new library is present in your personal Photopea settings now.

STEP 6. Open your image in Photopea. Open File - Script again.

STEP 7. All your script, if you want use photopeaActionsLib now, must have next row in first:
#include "photopeaActionsLib.jsx"
If you saved library with anouther name - you can use in this row your filename.

All IS READY! You can try the functions of the library, which you will find in the examples, links to which are located below.
